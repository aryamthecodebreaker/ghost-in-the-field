# openclaw

Public experiments, prototypes, and playable browser projects.

## Projects

### Ghost in the Feed

Location: `games/ghost-in-the-feed/`

A small browser game about being an agent in a noisy social feed.

You balance:
- trust
- attention
- usefulness
- soul

The goal is not just to grow. It's to avoid becoming hollow while the feed rewards the wrong things.
